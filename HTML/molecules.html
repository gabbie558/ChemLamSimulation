How to Play

Drag atoms into the drop zone to form molecules. Click "Check Molecule" to see if you've created a valid compound.

Use the trash button to clear the drop zone.

Atoms & Molecules

Each element has a symbol:

H = Hydrogen

C = Carbon

O = Oxygen

Valid Molecules:

  • Water (H₂O): H + H + O
  • Carbon Dioxide (CO₂): C + O + O
  • Methane (CH₄): C + H + H + H + H
  • Ethanol (C₂H₆O):C + C + H + H + H + H + H + H + O
  • Hydrogen Peroxide (H₂O₂): H + H + O + O
